About the Tools

We used the following tools to make this project:

    Client Side
  • Vue: we used Google's VueJS as our client-side JS framework for this responsive web app
  • Leaflet: we used the Leaflet API to display the main map with all of its features
  • Nominatim: we used the Nominatim API to go between name and location data
  • Foundation: we used Foundation CSS for styling
    Server Side
  • Express: we used the Express package to run the server
  • SQLite3: we used the SQLite3 package to make database queries

Fun Facts

  • There were roughly double forced entry, residental burgarlies in the day than at night (933 vs. 1715)
  • There were (only) 14,967 Narcotics incidents
  • There were about 18x Vandalism incidents compared to Graffiti (18,719 vs. 1,032)
  • About a third of the Homicides were committed in September
  • Payne-Phalen and Capitol River had the most Community Engagement Events and about a fourth of the total (869 and 716 out of 6649)
